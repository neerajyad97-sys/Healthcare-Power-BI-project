# Papollo Healthcare Dashboard

## 1. Project Title
Papollo Healthcare Analytics Dashboard | Patient Admissions & Hospital Performance Analysis

## 2. Short Description / Purpose
The Papollo Healthcare Analytics Dashboard is a Power BI solution designed to monitor hospital operations, patient admissions, disease trends, bed occupancy, doctor performance, diagnostic tests, patient feedback, and financial metrics. It provides healthcare administrators with actionable insights to improve patient care, optimize resource utilization, and enhance operational efficiency.

## 3. Tech Stack
This	dashboard was built by using the following tools and technologies:
* Power BI Desktop - Main data visualisation platform used for report creation.
* Power Query - Data transform and cleaning.
* DAX(Data Analysis Expression) - Used for calculated measures, dynamic visuals.
* File format - .pbix for development

## 4. Features / Highlights

### Business Problem
Healthcare organizations often struggle to consolidate patient, operational, and financial data into a single view. Without proper analytics, it becomes difficult to:
* Track patient admission trends.
* Monitor bed occupancy levels.
* Identify the most common diseases.
* Evaluate doctor workload and performance.
* Analyze diagnostic test demand.
* Measure patient satisfaction.
* Monitor billing and insurance coverage.

### Goal of the Dashboard
The dashboard aims to:
* Provide a centralized view of hospital performance.
* Track patient admissions and discharges over time.
* Analyze diagnosis patterns and disease prevalence.
* Monitor occupancy across General, Private, and ICU beds.
* Evaluate doctor-wise patient handling.
* Analyze diagnostic test utilization.
* Measure patient feedback and satisfaction levels.
* Track revenue generation and insurance coverage.

### Business Impact
* Improved hospital resource planning through occupancy analysis.
* Faster identification of high-frequency diseases and treatment trends.
* Better workforce allocation based on doctor workload insights.
* Enhanced patient experience through feedback monitoring.
* Increased financial visibility through billing and insurance analysis.
* Data-driven decision-making for hospital management.

### Dashboard Highlights
* 7,157 Patient Records analyzed.
* Disease-wise analysis covering Viral Infection, Flu, Malaria, Typhoid, and Pneumonia.
* Bed occupancy tracking across Private, General, and ICU categories.
* Diagnostic test analysis including Blood Tests, MRI, CT Scan, X-Ray, and Ultrasound.
* Doctor-wise performance and patient distribution monitoring.
* Patient satisfaction analysis using feedback ratings.
* Financial overview with:
    * Total Billing Amount
    * Insurance Coverage Amount
    * Revenue Trends
* Interactive filters for deeper analysis and decision-making.
